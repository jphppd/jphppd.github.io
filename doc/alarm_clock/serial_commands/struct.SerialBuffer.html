pub fn dequeue_command(&mut self) -> Result<Option<Command>, ()>

Try to dequeue a command from the input buffer. Return

  • Ok(Some()) when a command is identified
  • Err(()) when a \n separator was found, but no valid command could be parsed
  • Ok(None) most of the time, when there is no/not enough data Bytes are also removed from the ring buffer.
