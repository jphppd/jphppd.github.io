Generic boolean input.

Fields§

§pin: Pin<Input<PullUp>, PIN>

Pin to read the state from.

§logical_level_high: bool

Mapping between the electric level of the pin (+3.3V or +5V) and the logical level of the input.

§value: bool

Logical value of the input: true for an active state, false for an inactive state.
